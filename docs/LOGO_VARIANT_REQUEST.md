# Logo Variant Request – Message for Client

---

**Subject:** Request for Light Logo Variant for Website Header

Hi [Client Name],

To improve how the logo appears on our website, we need a second version of your logo optimised for dark backgrounds.

## What we need

**Light logo variant** – A version of your logo with **white or light-coloured text** (and icon/symbol) on a **transparent or dark background**.

## Why we need it

1. **Header behaviour** – Our site header is black at the top of the page and turns white when users scroll. We use two logo versions so the logo stays clear in both states.

2. **Current logo** – The logo you provided has black text on a white background. On the black header, the white background stands out and the black text is hard to see.

3. **Better visibility** – A light variant (white/light text) on the black header will be clear and readable without any white box.

4. **Professional look** – Many sites use separate logo variants for light and dark backgrounds. This keeps the logo legible and consistent across the site.

## Technical specs

- **Format:** PNG or SVG (transparent background preferred)
- **Content:** Same design as the main logo, but with white/light-coloured text and elements
- **Size:** Same proportions as the main logo; we will scale it for the header

## Where it will be used

- **Header (dark state)** – When the header is black (top of page)
- **Footer** – The footer has a dark background, so the light logo will be used there too for consistency
